ICE: In Class Exercises

Below are all the projects I completed in my WDD130 Web Fundamentals class during the Fall 2025 Semester. Each one challenged my skills and taught me how to use HTML and CSS in new and different ways.

Feel free to check them out via the links in each box!

1. Prophet Card

My first ICE assignment was to create an information card on our late prophet, President Russell M. Nelson.

I learned to follow a wireframe, read and write some basic HTML and CSS, and use a list through this project.

2. Apostle Spotlight

I created a spotlight card featuring each member of the First Presidency and Quorum of the Twelve Apostles.

This assignment taught me how to place images within divs and manipulate them so they look nice.

3. Favorite Devo Table

In theis ICE assignment, I made a table listing some BYUI devotionals I particularly liked. I had access to a table generator that assisted me, and now I know how to create tables using it on my own.

4. Halloween Page

This was a challenging assignment. My class was given an html with the framework we were to work with, and were then challenged to use our creativity to style the HTML without changing it. This was a real test of my creativity, but I feel pretty proud of how I did, especially with my level of ability.

5. Country Flags with Grid

This assignment was really fun for me, almost like a puzzle to solve. The objective was to use a premade set of colored squares, along with using a grid and creating gaps between elements, to create various national flags such as Denmark, Finland and Italy.

6. CSS Positioning Activity

This was also a fun, puzzle-like assignment. In this project, I made various shapes and patterns using CSS and some pregenerated colored squares. I learned how to overlay objects with precision during this assignment.

7. Image Gallery Issues

In this project, I resized the contents of an image gallery, which was full of oversized pictures of Christ. I learned how to resize images so they took up less space on my computer, then replaced the oversized files with my resized ones.

8. Signup Form Design

This was the last ICE challenge I completed in my Web Fundamentals class. I had to arrange the contents of a sign-up sheet to match a finished version. It challenged me a lot, but I managed to figure out how to position the elements the way I wanted them, and even figured out how to use shadows to make text look more distinctive and fancy.
